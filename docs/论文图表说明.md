## 1️⃣ 统计检验表格（Section 5: Results and Analysis）

### 📄 statistical_report_latex_table.tex

**文件位置**：`results/statistical_report_latex_table.tex`
**类型**：LaTeX表格代码
**论文章节**：5.4 Statistical Significance Tests

**内容包括**：
1. **Friedman Test Results** - 多算法显著性检验
   - χ² = 10122.0000
   - p-value < 0.001 (高度显著)
   - Critical Difference = 3.4052

2. **Average Ranks** - 算法平均排名
   - PSO: 1.0000 (第1名)
   - ICBO-Enhanced: 2.0000 (第2名)
   - GWO: 3.0000 (第3名)
   - ICBO: 4.0000 (第4名)
   - WOA: 5.0000 (第5名)
   - CBO: 6.0000 (第6名)
   - Random: 7.0000 (第7名/基线)

3. **Pairwise Comparisons** - 两两对比
   - CBO vs ICBO: p=0.0063 (显著改进)
   - CBO vs ICBO-E: p=0.0012 (高度显著改进)
   - ICBO vs ICBO-E: p=0.0064 (增强策略有效)

**论文撰写建议**：
```
The Friedman test results show highly significant differences among the seven
algorithms (χ²=10122.0, p<0.001). PSO achieved the best average rank (1.00),
followed by ICBO-Enhanced (2.00) and GWO (3.00). Pairwise comparisons using
Wilcoxon signed-rank tests reveal that ICBO-Enhanced significantly outperforms
CBO (p=0.0012, Cohen's d=0.1723).
```

## 4️⃣ 论文级对比图（Section 5: Main Results）

### 📊 algorithm_ranking_comparison.png

**文件位置**：`results/algorithm_ranking_comparison.png`
**尺寸**：178 KB
**分辨率**：300 DPI
**论文章节**：5.1 Overall Algorithm Performance

**图表内容**：
- 左侧：固定参数实验排名（5规模）
- 右侧：异构参数实验排名（7规模）
- 横向柱状图（排名越小越好）
- 7个算法对比

**关键发现**：
- 固定参数：ICBO-E排名1.60（超越PSO 2.20）
- 异构参数：PSO重回第一（1.71），ICBO-E第二（2.71）
- ICBO排名大幅提升：4.40 → 2.86

**论文描述建议**：
```
In the fixed-parameter experiment, ICBO-Enhanced achieves the best average rank
of 1.60, outperforming PSO (2.20). However, in the heterogeneous-parameter
experiment, PSO reclaims the top position with a rank of 1.71, while
ICBO-Enhanced ranks second at 2.71. Notably, ICBO shows significant improvement
(4.40 → 2.86), indicating that dynamic inertia weight is particularly beneficial
in diverse task environments.
```
